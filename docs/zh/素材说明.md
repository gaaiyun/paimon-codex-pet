# 素材说明

## 正式加载素材

Codex 实际加载的是：

- `pet/paimon/pet.json`
- `pet/paimon/spritesheet.png`

正式 spritesheet 是 8x9 atlas，每格 192x208，总尺寸 1536x1872。

## 扩展素材

为了便于展示和后续继续迭代，仓库里还包含：

- `assets/frames/`：从最终 atlas 抽出的透明单帧。
- `assets/expressions/`：精选表情单帧。
- `assets/supplemental/expression-sheet.png`：补充表情参考图。
- `assets/supplemental/action-sheet.png`：补充动作参考图。

这些素材不会被 `pet.json` 直接加载。它们适合用于 README、展示图、未来重制 atlas、或者手动挑选新动作。

## 设计方向

当前版本强调：

- 白色短发和金色小冠的第一眼识别度。
- 星形发饰、白蓝金配色和小披风。
- 小型 Codex pet 风格，而不是高精度插画。
- 更多“派蒙式”的表情倾向：开心、催促、惊讶、骄傲、委屈、思考。

为了避免加载器兼容问题，正式 atlas 没有增加额外行；新增动作以补充素材形式保存。
