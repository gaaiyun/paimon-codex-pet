# 安装与排错

## 安装步骤

在仓库根目录运行：

```powershell
Copy-Item -Recurse -Force .\pet\paimon "$env:USERPROFILE\.codex\pets\paimon"
```

然后重启 Codex，或在命令菜单里执行 `Force Reload Skills` / reload assets。

## 文件结构

```text
%USERPROFILE%\.codex\pets\paimon\
  pet.json
  spritesheet.png
```

`pet.json` 必须指向：

```json
{
  "spritesheetPath": "spritesheet.png"
}
```

## 已知兼容性

最初的版本使用 `spritesheet.webp`。部分 Codex 桌面版本可以在设置里发现 pet，但选择后无法成功加载。当前版本改为 PNG 作为正式加载文件，并保留 WebP 作为归档素材。

## 验证命令

```powershell
python "$env:USERPROFILE\.codex\skills\hatch-pet\scripts\validate_atlas.py" "$env:USERPROFILE\.codex\pets\paimon\spritesheet.png"
```

预期输出包含：

```text
"ok": true
"format": "PNG"
"mode": "RGBA"
"width": 1536
"height": 1872
```
