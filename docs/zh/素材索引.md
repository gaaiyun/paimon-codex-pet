# 素材索引

本仓库把“正式加载素材”和“扩展素材”分开保存。

## 正式加载素材

Codex 当前只加载：

- `pet/paimon/pet.json`
- `pet/paimon/spritesheet.png`

不要直接把补充动作塞进正式 atlas。正式 atlas 必须保持 8x9 布局，总尺寸 1536x1872，每格 192x208。

## 素材清单

机器可读索引：

- `assets/manifest.json`

这个文件记录：

- 正式加载包位置。
- 每个状态的帧数和单帧路径。
- 精选表情路径。
- 补充动作/表情素材表。
- 自动生成的预览拼图。

## 后续迭代建议

新增动作先作为补充素材保存到 `assets/supplemental/` 或 `assets/expressions/`。如果要升级正式 pet，需要重新生成完整 8x9 atlas，并重新执行：

1. `validate_atlas.py`
2. contact sheet 人工检查
3. 本地 Codex 安装目录同步
4. Codex 重启或 reload 后手动选择测试
