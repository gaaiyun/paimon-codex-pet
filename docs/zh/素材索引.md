# 素材索引

## 正式加载包

- `pet/paimon/pet.json`
- `pet/paimon/spritesheet.webp`

正式 atlas 为 1536×1872、8×9、单格 192×208，共 57 个有效帧。

## 可复用素材

- `assets/manifest.json`：机器可读索引。
- `assets/frames/`：九个状态的 57 个透明帧。
- `assets/expressions/`：8 个代表表情。
- `assets/gallery/`：状态和表情总览。
- `assets/supplemental/`：从本版素材重建的补充总览。
- `source/`：canonical base、prompt、生成请求和 job provenance。
- `qa/previews/`：当前 WebP 版本的九个 GIF。

状态帧数：`idle` 6、`running-right` 8、`running-left` 8、`waving` 4、`jumping` 5、`failed` 8、`waiting` 6、`running` 6、`review` 6。
